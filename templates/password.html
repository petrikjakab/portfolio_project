My dumb password

During this project I learned about how to use APIs with the requests library and how hashing works, what is it used for.

This script uses the API of Have I Been Pwned? and checks whether your password has ever been involved in a data breach.
In order to not send our password directly to the API the program will convert it to a hash value using the sha1 algorithm and only sends the first and last five characters to the API and checks if there are any hits in their wast database, then prints out a suggestion to change it or not.


You can download the code itself from my github and run it from the command line.
