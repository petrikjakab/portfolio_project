This Website

For many years I always revisited web design and tinkered in HTML & CSS, but never deployed a site. Mainly because I never reached to the point where I needed to deal with the back-end, well, in 2020 this changed, since on the ZTM python course one of the task was to create a website and deploy it with a working contact form.

The main task this time was to build a server using the flask library and deploy a website based on a template. After reading the quickstart guide of flask and some searching I managed to get the web server up and running with the development mode turned on.

After a few days I hacked together the site from 3 templates I found and wrote some words about my projects, pushing it to github at the end of every day so I will gain some experience using it.
On the ZTM course we also set up a contact form, once we finished with the code, we connected the html tags to it and hooray! I have completed my site, Awesome!

Next thing to show it off! Ohh wait, I still haven't deployed any website..bummer.

Back to the course. Some time later I understood how to push my web app with git to pythonanywhere.com, which I can use to host my first web app for free.
I also learned how to make a requirements.txt with the pip freeze command that basically captures all the installed packages in the environment, so for example if I create something using python and want to deploy it elsewhere I can know what packages are required to do so.
Last thing was to create a repository for the project, push it to github and then clone it to the pythonanywhere server. We are up and running baby!
Hope you are satisfied with the end results, just like me. This was a cool project!

Cheers,
Petrik
